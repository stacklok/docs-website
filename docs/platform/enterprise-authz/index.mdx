Enterprise authorization lets cluster admins express MCP access control in
familiar terms, without learning a domain-specific policy language.

Namespace owners can grant access to MCP servers they own, so the platform team
isn't in the loop on every change.

[Learn more about Stacklok Enterprise](../index.mdx).

:::
