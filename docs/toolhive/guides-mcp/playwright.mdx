## Overview

The [Playwright MCP server](https://github.com/microsoft/playwright-mcp) enables
automated web testing and interaction using the
[Playwright](https://playwright.dev/) framework. It allows users to script
browser actions, perform end-to-end testing, and automate web-based workflows
across multiple browsers.
