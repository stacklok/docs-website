Virtual MCP Server (vMCP) aggregates multiple backend MCP servers into a single
endpoint, enabling unified tool access, centralized authentication, and
multi-step workflows.

## When to use Virtual MCP

- Managing multiple MCP servers that should appear as one
- Centralizing authentication across backends
- Creating reusable workflows spanning multiple systems
