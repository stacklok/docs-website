## thv secret provider

Configure the secrets provider

### Synopsis

Configure the secrets provider.
Valid secrets providers are:
  - encrypted: Encrypted secrets provider
  - 1password: 1Password secrets provider (currently only supports getting secrets)

```
thv secret provider <name> [flags]
```
