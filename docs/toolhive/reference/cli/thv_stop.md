## thv stop

Stop an MCP server

### Synopsis

Stop a running MCP server managed by ToolHive.

```
thv stop [container-name] [flags]
```

### Options

```
  -h, --help          help for stop
      --timeout int   Timeout in seconds before forcibly stopping the container (default 30)
```
