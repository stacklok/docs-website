This is the JSON schema for the ToolHive-native registry format. It defines the
structure and constraints for registry entries, ensuring that all entries
conform to a consistent format.

:::info

This format is considered legacy now that an
[official upstream registry format](registry-schema-upstream.mdx) exists. The
ToolHive-native format continues to work for the built-in registry and for
custom file-based registries used by the CLI and UI. The
[Registry Server](/toolhive/guides-registry/intro) only accepts the upstream
format.

:::

To use this schema in your own custom registry file, add a `$schema` property at
the top of your JSON file:

```json
{
  "$schema": "https://raw.githubusercontent.com/stacklok/toolhive-core/main/registry/types/data/toolhive-legacy-registry.schema.json",
  ...
}
```
